# LineByLine

## 실행 방법

### 1. 패키지 설치

```bash
cd LineByLine
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

터미널에 표시되는 URL로 브라우저에서 접속합니다.

### 3. 프로덕션 빌드

```bash
npm run build
npm start
```

## 스크립트

| 명령어 | 설명 |
|--------|------|
| `npm run dev` | 개발 서버 실행 |
| `npm run build` | 프로덕션 빌드 |
| `npm start` | 프로덕션 서버 실행 |
| `npm run check` | TypeScript 타입 체크 |
| `npm run db:push` | DB 스키마 푸시 |
